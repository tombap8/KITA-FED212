[ HTML 기초 ]

1. HTML은 Hyper Text Markup Language를 나타냅니다.

: 하이퍼텍스트(Hyper Text)란?
- 순서가 없는 정보를 담고 있는 텍스트

: 마크업(Markup) 이란?
- 정보를 구조화하여 순서대로 웹 페이지에 띄우는것

-> Hyper Text Markup Language란?
순서가 없는 정보를 담고 있는 텍스트의
 정보를 구조화하여 순서대로 웹 페이지에
표시하기 위한 브라우저 언어

[ 구조화란? ]
- 구조 -> 체계 -> 순서/분류
-> 체계가 없는 데이터를 체계있게 순서를 정해주는 것
