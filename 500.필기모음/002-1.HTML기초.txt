[ HTML 기초 ]

1. HTML은 Hyper Text Markup Language를 나타냅니다.

: 하이퍼텍스트(Hyper Text)란?
- 순서가 없는 정보를 담고 있는 텍스트

: 마크업(Markup) 이란?
- 정보를 구조화하여 순서대로 웹 페이지에 띄우는것

-> Hyper Text Markup Language란?
순서가 없는 정보를 담고 있는 텍스트의
 정보를 구조화하여 순서대로 웹 페이지에
표시하기 위한 브라우저 언어

[ 구조화란? ]
- 구조 -> 체계 -> 순서/분류
-> 체계가 없는 데이터를 체계있게 순서를 정해주는 것

2. HTML 은 XML에서 온 것이다!
- XML 은 정보를 구조화하는 확장가능한 마크업 언어이다!
- 표기법:
    1) 최상단에 xml선언부
    2) 최상위 태그가 단 하나임! -> Root라고함!
    3) 하위 구조화태그 생성

    예)
    <?xml version="1.0" encoding="UTF-8"?>
    <회사정보>
        <회사 창립일="1988년">
            <회사명>유아이</회사명>
            <사원수>3000</사원수>
            <연매출>1억</연매출>
        </회사>
        <회사 창립일="2000년">
            <회사명>데이터굿</회사명>
            <사원수>400</사원수>
            <연매출>10억</연매출>
        </회사>
    </회사정보>


(참고: 파일확장자란?)
- 파일 포멧을 알려주는 표시자
※ 파일 확장자를 반드시 보이게할것!

3. HTML의 태그구조
- 태그: 구조화 요소의 표시자
- 태그(Tag)는 요소(Element)라고도 함

1)  선언부
-> 느껴! 문서타입 HTML의
<!doctype html>
-> 브라우저가 이것을 읽는 순간
html5문법으로 읽을 준비를 한다!

2) 전체부모(html) : 오직하나!
-> 이것을 다른말로 Root라고 함
<html></html>

3) 머리(head)
-> 웹 페이지 타이틀, 인코딩설정 등 메타데이터(요약데이터)
3-1) meta태그: 문서의 기본정보
    - charset="utf-8"
    -> 한글, 일본어, 한자어 등 2byte 체계의 문자셋이
    브라우저에 잘 표시되도록 인코딩 설정함!

    - 사이트의 기본 정보 등을 표시함

3-2) title태그 : 브라우저 탭에 제목출력!
